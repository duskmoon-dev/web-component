# @duskmoon-dev/echarts-el

Provides html custom elements: 

- `dm-echarts`
- `dm-echarts-gl`

```html
<dm-echarts>
    {JSON.stringify(config)}
</dm-echarts>

<dm-echarts-gl>
    {JSON.stringify(config)}
</dm-echarts-gl>
```

## For developer

To install dependencies:

```shell
bun install
```

To Show storybook:

```shell
bun run storybook
```

To build:

```shell
bun run build
```

To publish:

```shell
npm publish
```
